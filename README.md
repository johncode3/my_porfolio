# Sovann Kakkada Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript.

## Project Structure

- `index.html` - main landing page
- `projects/` - case study pages
- `css/` - modular stylesheet structure
- `js/` - client-side interactions
- `assets/` - images and icons

## Run Locally

Open `index.html` in a browser, or use VS Code Live Server for a smoother workflow.

## GitHub Workflow

1. Make your changes.
2. Check the result locally.
3. Commit and push to your GitHub repository.

## Notes

The CSS is split into base, layout, component, and utility files to keep future edits easier to manage.
